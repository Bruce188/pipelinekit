# commands/

Project-scoped slash commands. Empty by default in the portable build.

Skills live in `../skills/` and are the primary user-invocable surface (e.g., `/pipeline`, `/review`, `/ppr`, `/create-plan`).

Use this directory for one-off project commands. Format: each file is a markdown file with frontmatter; see Anthropic docs at https://docs.claude.com/claude-code/slash-commands.
